## Description

[Kodo](https://github.com/chandanpasunoori/kodo-search-sample) Search Sample

Sample project with Nest.js microservices using NATS as bridge.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# search service
$ npm run start search-service

# kodo service
$ npm run start kodo-service

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
